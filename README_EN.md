# Wang Yue Personal Website

[中文](README.md) | English

A personal showcase website for Wang Yue, a student at Wenzhou University, featuring an Apple-inspired design.

## Tech Stack
- **Frontend**: Vue.js 3 + Vite + Tailwind CSS
- **Backend**: Node.js + Express.js
- **Database**: MySQL
- **Deployment**: 
  - Frontend: GitHub Pages
  - Backend: Railway/Render (free services)

## Project Structure
```
wangyue-website/
├── frontend/          # Vue.js frontend
├── backend/           # Node.js backend
├── database/          # MySQL database scripts
└── docs/             # Project documentation
```

## Features
1. **Home** - Personal introduction and highlights
2. **About** - Detailed personal info and education background
3. **Projects** - Wenzhou University project showcase
4. **Skills** - Technical capabilities display
5. **Contact** - Contact information

## Design Style
- Apple-inspired clean and modern design
- White as primary color with elegant animations
- Responsive layout with mobile optimization

## Quick Start
```bash
# Frontend development
cd frontend
npm install
npm run dev

# Backend development
cd backend
npm install
npm start
```
