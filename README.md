# 王越个人展示网站

温州大学学生王越的个人展示网站，采用苹果风格设计。

## 技术栈
- **前端**: Vue.js 3 + Vite + Tailwind CSS
- **后端**: Node.js + Express.js
- **数据库**: MySQL
- **部署**: 
  - 前端: GitHub Pages
  - 后端: Railway/Render (免费服务)

## 项目结构
```
wangyue-website/
├── frontend/          # Vue.js 前端
├── backend/           # Node.js 后端
├── database/          # MySQL 数据库脚本
└── docs/             # 项目文档
```

## 功能模块
1. **首页** - 个人简介、亮点展示
2. **关于** - 详细个人介绍、教育背景
3. **项目** - 温州大学项目展示
4. **技能** - 技术能力展示
5. **联系** - 联系方式

## 设计风格
- 借鉴苹果官网的简洁现代设计
- 白色为主色调，优雅动画
- 响应式布局，移动端优化

## 快速开始
```bash
# 前端开发
cd frontend
npm install
npm run dev

# 后端开发
cd backend
npm install
npm start
```
